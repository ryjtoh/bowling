The slope formula y = mx + b is a crucial concept to grasp in algebra. It tells you a straight (linear) line's behavior and is a formula for finding point on that line. Here's a rundown of the variables in the formula: ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ x : tells the horizontal position on the coordinate system (left/right) y : tells the vertical position on the coordinate system (up/down) m : the slope, which tell you how steep the line is if you can imagine the positive y direction to be pointing toward the sky! b : the y-intercept, where the straight line hit the y-axis (the line on the vertical axis) ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ Now that you know a bit about the terminology of the slope formula, lets visualize it using this interactive coordinate system. - Use the slope slider to control the steepness of the line - Use the y-intercept to control where the line hits the y-axis ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ Instructions: There are going to be 2 golf holes on the coordinate system that fit a particular linear line. 2 golf balls will spawn from the "middle" of the line, so plan carefully! Your goal is to figure out what line fits using the sliders, then press the swing button to see if you get two hole in ones!
